Prerequisites:
o	Angular CLI: 9.0.1
o	Node: 12.15.0

Installation Guide:
o	npm install -g @angular/cli

To Install necessary dependencies follow below commands 
o	npm install 
o	npm install -g typescript ts-node
o	npm install --save @types/express express body-parser  nodemon
o	npm i @types/cors
o	npm i n-readlines

Start the server
o	“npm start”
